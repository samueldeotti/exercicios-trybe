O primeiro passo para o versionamento é: criar a pasta, entrar na pasta, usar o comando git init, criar um arquivo, fazer a modificação no arquivo, usar o git add . e depois git commit -m 'Mensagem'.
Para criar uma nova branch, usar o gamando git checkout -b nova-branch, quando estiver na nova branch, fazer alterações nos arquivos e ver se ta tudo certo para subir para a branchmain, se estiver tudo certo, dar um git add . depois git commit -m 'Mensagem', apos isso, voltar para a branchmain, checkout nome-da-branchmain, apos isso dar um git merge nome-da-branch-que-estava-sendo-usada.
